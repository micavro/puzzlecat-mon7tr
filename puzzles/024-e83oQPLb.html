题面

  • 我 + 三道杠 = ?（三字）
  • 我 + 正电荷 = ?（四字）
  • 我我 = ?（两字）
  • 标题：绿泡泡。
  • Flavor 要求一行提交三个答案，以空格分隔；有三个里程碑。

机制

“绿泡泡”是微信的常见代称。三个 Cryptic 不是把“我”按字义做代数，而是把它看成一个单人轮廓图标，再与右侧符号组合，读微信界面中的正式功能名称。

  1. 单人 + 三条列表线 = 通讯录（三字） - 微信底部的“通讯录”图标是联系人/人物配合列表线条的视觉意象。
  2. 单人 + 正电荷符号 + = 新的朋友（四字） - 微信通讯录中的“新的朋友”入口以人物加号图标表示。
  3. 双人 = 群聊（两字） - 两个人物轮廓是群组/群聊的常见图标，且“群聊”是微信通讯录中的正式二字入口。

第一项先指向“通讯录”，后两项正好是同一通讯录界面中的条目，构成统一层级，而非三个任意的微信近义词。

候选排序

  1. 通讯录 新的朋友 群聊 — 高置信（约 88%）。三个字数完全吻合，图标机制一致，后两项又同属第一项页面。
  2. 通讯录 添加朋友 群聊 — 中低置信。人物加号也可描述“添加朋友”，且同为四字；但微信通讯录列表的正式入口通常显示“新的朋友”，因此次选。
  3. 朋友圈 添加朋友 朋友 — 很低置信。能勉强套微信语境，但“三道杠”和双人图标对应不精确，也缺乏统一界面层级。
